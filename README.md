# pq

Command-line Parquet file viewer written in Rust.

## Status

- ✅ Schema command - done
- ✅ Head command - done
- ✅ Columns command - done
- ✅ Count command - done

## Features

- Show schema (column names and data types)
- Show first N rows (like `head`)
- Show only column names
- Count total records

## Installation

```bash
cargo install pq
